# cristobal.dev
[cristobal.dev](http://cristobal.dev) web source code.

## Install

```bash
git clone --recurse-submodules -j8 git@github.com:cristobalcl/cristobalcl-src.git
cd cristobalcl-src
```

Install [Hugo](https://gohugo.io/).

Mac:

```
brew install hugo
```

## Update web

```bash
./deploy.sh
```
